### 两点说明

- 脚本运行后会设置屏幕亮度为0，媒体音量为0，是正常现象，手动停止脚本会重置音量和亮度为开始运行脚本之前的状态。
- 脚本每次运行时需要给无障碍权限(模拟点击，滑动等操作)，第一次运行时需要修改系统设置权限(修改屏幕亮度和音量)
